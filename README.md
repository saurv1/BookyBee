BookyBee
BookyBee is a comprehensive web-based service marketplace designed to help users manage and
access professional home services easily. The system connects customers with verified experts for
various household needs, including cleaning, repairs, painting, and more, all through a streamlined
and intuitive platform.

Project Objective
The main objective of this project is to develop a reliable system that bridges the gap between
service providers and customers. By providing a simple, efficient, and secure interface, BookyBee
aim to solve the hassle of finding trustworthy local experts while ensuring a high-quality user
experience.

Features
o User Registration & Authentication: Secure login and signup system for both
Customers and Service Providers.
o OTP Verification: Enhanced security using One-Time Passwords for account
verification.
o Service Discovery: Advanced search and location-based filtering to find the perfect
expert near you.
o Real-time Communication: Integrated chat system for direct messaging between users
and providers.
o Booking Management: Easy scheduling and tracking of service requests with instant
confirmations.
o Admin Dashboard: Comprehensive management tool for overseeing users, service
providers, and commission-based revenue.
o Review & Rating System: Transparent feedback system to help users find the best-rated
professionals.
o Secure payment gateways: For payment esewa and Khalti test api is used.
o Invoice: For each payment invoice is generated.
o Secure Logout System: Ensuring user data privacy and session security.

Technologies Used
Frontend
React.js (Vite): For building a fast and responsive user interface.
Tailwind CSS: For modern, utility-first styling.
Framer Motion: For smooth micro-animations and transitions.
Redux Toolkit: For efficient global state management.
Recharts: For data visualization in the admin and provider dashboards.
Lucide React: For a consistent and clean iconography.
Backend
Node.js & Express.js: For a scalable and robust server-side architecture.
MongoDB & Mongoose: For flexible and efficient data storage and object modeling.
JSON Web Token (JWT): For secure, stateless authentication.
BcryptJS: For advanced password hashing and security.
Cloudinary & Multer: For seamless image and media file management.
Nodemailer: For automated email notifications and OTP delivery.

Installation and Setup
Steps to run the project locally.
1. Clone the repository
https://github.com/saurv1/BookyBee
2. Go to the project folder
cd backend
3. Install required dependencies
npm init -y
npm I express nodemon cors mongoose
4. Go to the project folder
cd frontend
5. Install required dependencies
Npm create vive@latest
Npm i react-router-dom
6. Run the application
Backend: npm start
Frontend: npm run dev
